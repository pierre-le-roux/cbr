Coffee Barista Recipies (BCR)

Tables:
machine
grinder
purchase
coffee
roastery
recipy


machine:
    id (auto-increment)
    name (str)
    brand (str)

grinder:
    id (auto-increment)
    name (str)
    brand (str)

roastery:
    id  (auto-increment)
    name (str)
    country

purchase:
    id
    coffee_id
    roasted date
    purchhased date
    
coffee:
    id (auto-increment)
    name (str)
    roast level (light, dark)
    origins (columbia, costa-rica, etc) [multiple - indicate blend]
    processing method (wet/washed, dry/natural, honey & pulped natural)
    species (Arabica, etc)
    variety (caturra, catua√≠, bourbon, typica, etc)
    elevation (m.a.s.l.)
    taste notes (tangerine, dries fruit, cocoa)
    roastery_id (foreign key)

recipe:
    id (auto-increment)
    machine_id
    grinder_id
    purchase_id
    timestamp
    grind
    dose (float - grams) [10-24]
    yield (float - grams)
    ratio (calculated) [1:2]
    brew time (push of the button until desired yield)
    temperature
    water
    milk
    rating (float) [1-5]
